# ScreenPal 屏幕学伴

> 你的图书馆AI学习搭子 — 实时读屏、语音对话、自动生成复习笔记

**ScreenPal** 是一款为大学生期末复习设计的桌面AI工具。它能实时捕获屏幕上的课件内容，通过多模态大模型理解课件，和你用语音/文字自然对话讨论学习内容，并根据你的疑难点自动生成复习笔记和模拟题。

## ✨ 功能

- **实时屏幕理解** — 一键捕获屏幕课件，AI 秒懂你的学习内容
- **语音对话** — 免提模式/按键说话，像和图书馆对面的朋友聊天一样自然
- **主动引导学习** — AI 不只是答问题，会主动追问、确认理解、引导思考
- **4种学伴人格** — 图书馆死党 / 辅导老师 / 耐心教练 / 思辨搭档
- **疑难点检测** — 自动识别你卡住的地方，提醒整理笔记
- **复习笔记生成** — AI 根据对话上下文生成结构化笔记（含记忆锦囊+速测题）
- **截图收藏** — 重要课件一键收藏，期末回顾
- **导出复习手册** — 所有笔记合并导出为完整的 Markdown 复习文档
- **框选捕获区域** — 拖拽框选只截课件区域，不截桌面其他内容
- **迷你模式** — 收缩为 240×48 像素条，不挡课件

## 🖥️ 双模式

| 模式 | 启动方式 | 特点 |
|------|---------|------|
| **网页版** | `python app.py` → 浏览器 | 语音功能完整，三栏布局 |
| **浮动组件** | `python widget.py` 或网页内点击"组件" | 桌面置顶小窗，不占屏幕空间 |

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动网页版
python app.py
# 浏览器打开 http://127.0.0.1:5000

# 3. 或启动浮动组件
python widget.py
```

首次使用需在设置中配置 AI 服务（推荐智谱 GLM-4V，免费额度够日常使用，国内直连）。

## 🤖 支持的 AI 后端

| 后端 | 模型 | 费用 | 国内访问 |

## 📁 项目结构

```
ScreenPal/

├── app.py              # Flask 网页服务器
├── widget.py           # 浮动小组件（tkinter原生窗口）
├── templates/
│   └── index.html      # 网页版前端（语音对话+免提模式）

```

## 🛠️ 技术栈

- **屏幕捕获**: mss + Pillow
- **AI 后端**: 智谱 GLM-4V / Gemini / Claude（多模态视觉理解）
- **语音识别**: Web Speech API（网页）/ speech_recognition（组件）
- **语音合成**: Web Speech API / pyttsx3
- **前端**: Flask + 原生 HTML/CSS/JS
- **桌面组件**: tkinter（Python 内置）
